적용 위치 AI 분석 > 학급 분석
01

기획 요약

학기말마다 반복적으로 발생하는 학급 데이터 요청을 줄이기 위해, 교사가 AIDT에서 담당하는 반의 월별 학급 데이터를 엑셀로 직접 다운로드할 수 있도록 제공

기능 목적

관리자 개입 없이 사용자가 필요한 시점에 직접 데이터를 받도록 하여 반복 CS를 줄이고, 교사 셀프서비스 강화

적용 위치

AI 분석 > 학급 분석에 배치

핵심 방향

기존의 교사 ID 조회는 제거하고, 로그인한 교사 기준으로 반 목록을 자동 조회함. 현재 진입한 반은 기본 선택값으로 노출

02

AS-IS / TO-BE 비교

현재 관리자용 구조를 교사 화면 맥락에 맞게 재구성

AS-IS

자사 관리 시스템

  • 관리자가 사용자의 데이터 추출 기능
  • 교사 ID 입력 후 조회 필요
  • 한 번에 여러 교사 ID 조회 가능
  • 조회된 교사 현황 확인 후 선택 다운로드
  • 학급 데이터가 없으면 상태값으로 다운로드 불가 노출
TO-BE

AIDT 교사 화면

  • 교사가 AIDT에서 직접 다운로드하는 기능
  • 교사 ID 입력/조회 영역 제거
  • 로그인한 교사 기준으로 담당하는 반 자동 조회
  • 현재 AIDT 진입 시 선택한 반 기본 선택
  • 필요한 반만 교사가 추가 선택하여 엑셀 다운로드

04

정책 및 예외 처리

사용자 조회 제거, 현재 반 기본값, 다중 반 선택, 다운로드 불가 상태를 명확히 정의

구분 정책 화면 반영 방식
사용자 조회 교사 ID 입력/조회 기능 제거 로그인한 교사 기준으로 담당 반 자동 조회
기본 선택값 현재 AIDT 진입 시 선택한 반을 기본값으로 설정 현재 반 체크박스 기본 선택 및 ‘현재 선택 반’ 뱃지 노출
다중 반 선택 교사가 여러 반을 담당하는 경우 다른 반도 선택 가능 담당 반 목록 전체 노출 및 체크박스 제공
다운로드 불가 학급 데이터가 없는 반은 다운로드 불가 상태값 ‘다운로드 불가’, 체크박스 비활성화, 회색 행 처리
다운로드 실행 다운로드 가능한 반이 1개 이상 선택된 경우에만 실행 가능 엑셀 다운로드 버튼 활성/비활성 처리
